## Basic Encryption & Decryption

# Project Overview

This project was completed as part of the DecodeLabs Cyber Security Internship Program. The objective of this project was to understand the fundamentals of encryption and decryption by building a simple Caesar Cipher application using Python. The program accepts user input, encrypts the text using a shift-based cipher, and then decrypts it back to its original form.

While working on this project, I gained practical experience with basic cryptography concepts and learned how encryption helps protect information from unauthorized access. Developing this project also strengthened my Python programming skills and gave me a better understanding of how security concepts can be implemented through code. It was a valuable hands-on exercise that connected cybersecurity theory with real programming practice.

# Features

- Encrypts user-provided text using the Caesar Cipher technique
- Decrypts encrypted text back to its original form
- Displays both encrypted and decrypted outputs
- Simple and beginner-friendly implementation
- Demonstrates the fundamentals of data confidentiality

# How to Run the Project

1. Clone or download this repository.
2. Open the project folder in Visual Studio Code.
3. Open the terminal.
4. Run the following command:

python encryption_decryption.py

5. Enter the text you want to encrypt.
6. View the encrypted and decrypted output generated by the program.
